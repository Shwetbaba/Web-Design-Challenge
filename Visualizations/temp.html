Max Temperature

The purpose of this project was to to analyze how weather changes as you get closer to the equator. To accomplish this analysis, we first pulled data from the OpenWeatherMap API to assemble a datasetover 500 cities .

After assembling the dataset, we used Matplotlib to plot Max tempreature vs. latitude. Which shows that as latitude increases tempreature keep decreasing and vice versa. In other words it is showing inverse repationship between tempreature and latitude.
